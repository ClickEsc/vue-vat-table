# Компонент-таблица (Vue3)

## Демо
https://clickesc.github.io/vue-vat-table/

#### Возможности:
- поле в колонке "Цена, руб. с НДС" расчитывается автоматически по формуле на основе данных в полях "Цена, руб. без НДС" и "НДС, %"

При изменении поля в строке объект строки с текущим состоянием выводится в console.log  

## Запуск проекта

```bash
  npm i && npm run dev
```
